Fonaments de Programació I i II

Conceptes fonamentals de programació explicats amb definicions i exemples pràctics en JavaScript.

Què és la programació?

La programació és el procés d'escriure instruccions que un ordinador pot seguir per realitzar una tasca. És la base de tot el programari que fem servir cada dia.

Definició clau

Programar consisteix a dissenyar, escriure i mantenir el codi font de programes informàtics. El codi s'escriu en llenguatges com JavaScript, Python o Java.

Exemple: primer programa

// Mostra un missatge a la consola
console.log('Hola, món!');

// Demana el teu nom i et saluda
let nom = prompt('Com et dius?');
console.log('Hola, ' + nom + '!');

Aquest exemple mostra com donar instruccions bàsiques a l'ordinador.

Variables i tipus de dades

Les variables emmagatzemen informació a la memòria. Cada variable té un tipus de dada que determina quina classe de valor pot contenir.

Tipus bàsics

  • number: números enters i decimals
  • string: text
  • boolean: vertader o fals
  • undefined: variable sense valor assignat

Exemple en JavaScript

let edat = 25;              // number
let preu = 19.99;           // number decimal
let nom = 'Anna';           // string
let esEstudiant = true;     // boolean

console.log(typeof edat);   // Mostra "number"
console.log(nom + ' té ' + edat + ' anys');

Es fan servir les paraules clau let o const per declarar variables.

Operadors

Els operadors permeten realitzar operacions amb valors i variables.

Tipus d'operadors

  • Aritmètics: +, -, *, /, %
  • Comparació: ==, ===, >, <, >=, <=
  • Lògics: &&, ||, !

Exemple en JavaScript

let a = 10, b = 3;

// Operadors aritmètics
console.log('Suma:', a + b);        // 13
console.log('Mòdul:', a % b);       // 1

// Operadors de comparació
console.log('a és major que b?', a > b);  // true

// Operadors lògics
let majorEdat = true;
let tePermis = false;
console.log(majorEdat && tePermis);       // false

Condicionals

Les estructures condicionals permeten executar diferents blocs de codi segons una condició.

Estructures disponibles

  • if / else if / else: per a condicions múltiples
  • switch: per a múltiples valors fixos

Exemple en JavaScript

let nota = 85;

if (nota >= 90) {
  console.log('Excel·lent');
} else if (nota >= 70) {
  console.log('Notable');
} else if (nota >= 50) {
  console.log('Aprovat');
} else {
  console.log('Suspès');
}

// Operador ternari
let missatge = nota >= 50 ? 'Aprovat' : 'Suspès';
console.log(missatge);

L'operador ternari condició ? valor1 : valor2 és una forma compacta d'if/else.

Bucles

Els bucles repeteixen un bloc de codi mentre es compleixi una condició.

Tipus de bucles

  • for: quan saps quantes vegades repetir
  • while: mentre es compleixi una condició
  • do...while: executa almenys una vegada

Exemple en JavaScript

// Bucle for: imprimeix els números de l'1 al 5
for (let i = 1; i <= 5; i++) {
  console.log('Número:', i);
}

// Bucle while: suma fins a arribar a 20
let suma = 0, comptador = 1;
while (suma < 20) {
  suma += comptador;
  comptador++;
}
console.log('Suma total:', suma);

Funcions

Les funcions agrupen instruccions reutilitzables que realitzen una tasca específica.

Parts d'una funció

  • Declaració: es defineix amb function
  • Paràmetres: valors d'entrada
  • Return: valor que retorna la funció

Exemple en JavaScript

function sumar(a, b) {
  return a + b;
}

function esParell(numero) {
  return numero % 2 === 0;
}

console.log('La suma és:', sumar(8, 5));        // 13
console.log('8 és parell?', esParell(8));        // true

// Funció fletxa (arrow function)
let doble = (x) => x * 2;
console.log('El doble de 7 és:', doble(7));      // 14

Les funcions fletxa => són una sintaxi més moderna i concisa.

Arrays

Un array és una col·lecció ordenada d'elements que s'emmagatzema en una sola variable.

Característiques

  • S'hi accedeix a cada element per l'índex (comença a 0)
  • Poden contenir qualsevol tipus de dada
  • Tenen mètodes útils com push, pop, length

Exemple en JavaScript

let fruites = ['poma', 'plàtan', 'taronja'];

console.log(fruites[0]);            // "poma"
console.log('Total:', fruites.length);  // 3

fruites.push('raïm');               // Afegeix al final
fruites.pop();                       // Treu l'últim

// Recórrer un array
for (let i = 0; i < fruites.length; i++) {
  console.log('Fruita:', fruites[i]);
}

POO bàsica

La Programació Orientada a Objectes (POO) organitza el codi en classes i objectes.

Conceptes clau

  • Classe: motlle o plantilla per crear objectes
  • Objecte: instància d'una classe
  • Constructor: mètode que s'executa en crear un objecte

Exemple en JavaScript

class Persona {
  constructor(nom, edat) {
    this.nom = nom;
    this.edat = edat;
  }

  presentarSe() {
    return 'Hola, sóc ' + this.nom + ' i tinc ' + this.edat + ' anys.';
  }
}

let estudiant = new Persona('Lluís', 22);
console.log(estudiant.presentarSe());

Les classes permeten crear múltiples objectes amb la mateixa estructura i comportament.
