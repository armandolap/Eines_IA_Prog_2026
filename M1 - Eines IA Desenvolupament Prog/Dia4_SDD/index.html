Programació

La programació és el procés d'escriure instruccions que un ordinador pot entendre i executar. Un programa és una seqüència ordenada d'instruccions que resol un problema determinat.

Conceptes clau: algoritme, instrucció, programa, llenguatge de programació.

// El primer programa
console.log("Hola, món!");

// Missatge amb variables
let nom = "Anna";
console.log("Hola, " + nom + "!");

Variables

Una variable és un espai a la memòria que guarda un valor. Ens permet emmagatzemar dades i referir-nos-hi amb un nom descriptiu.

Tipus de dades: number, string, boolean, undefined, null.

let nom    = "Maria";
let edat   = 22;
let esMajor = true;

console.log(nom + " té " + edat + " anys.");

Condicionals

Les estructures condicionals permeten executar diferents blocs de codi segons si una condició és certa o falsa.

Estructures: if, else if, else.

let temperatura = 30;

if (temperatura > 25) {
  console.log("Fa calor");
} else if (temperatura > 15) {
  console.log("Temperatura agradable");
} else {
  console.log("Fa fred");
}

Bucles

Un bucle repeteix un bloc de codi mentre es compleix una condició. Evita escriure el mateix codi moltes vegades.

Tipus: for, while, do while.

// Comptar de l'1 al 5
for (let i = 1; i <= 5; i++) {
  console.log("Número " + i);
}

// Mentre
let x = 0;
while (x < 3) {
  console.log("x = " + x);
  x++;
}

Funcions

Una funció agrupa un conjunt d'instruccions sota un nom per reutilitzar-lo tantes vegades com calgui, evitant repetir codi.

Parts: nom, paràmetres, cos, valor de retorn (return).

function salutacio(nom) {
  return "Hola, " + nom + "!";
}

function suma(a, b) {
  return a + b;
}

console.log(salutacio("Pau"));
console.log("2 + 3 = " + suma(2, 3));

Arrays

Un array és una col·lecció ordenada d'elements. Cada element s'identifica per la seva posició (índex), que comença a 0.

Operacions: accedir, afegir, modificar, recórrer.

let colors = ["vermell", "verd", "blau"];

console.log(colors[0]);    // "vermell"
console.log(colors.length); // 3

for (let i = 0; i < colors.length; i++) {
  console.log("Color: " + colors[i]);
}

Programació Orientada a Objectes

La POO organitza el codi en classes i objectes. Una classe és una plantilla; un objecte n'és una instància amb estat (propietats) i comportament (mètodes).

Conceptes: classe, objecte, propietat, mètode, constructor.

class Persona {
  constructor(nom, edat) {
    this.nom = nom;
    this.edat = edat;
  }

  presentar() {
    return "Sóc " + this.nom +
           " i tinc " + this.edat + " anys.";
  }
}

let joan = new Persona("Joan", 25);
console.log(joan.presentar());
