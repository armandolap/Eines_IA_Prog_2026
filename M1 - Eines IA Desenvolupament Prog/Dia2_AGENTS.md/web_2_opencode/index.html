Fonaments de Programació I i II

Apunts estàtics amb definicions clau i exemples de codi interactius en JavaScript.

Programació I: Estructures bàsiques i seqüència

La programació estructural organitza el codi mitjançant blocs lògics, evitant salts desordenats i facilitant la comprensió del flux.

La seqüència

Les instruccions s'executen de dalt a baix, de manera ordenada. Una variable es pot declarar i utilitzar posteriorment en la seqüència.

Exemple: càlcul d'un descompte

// Declarem les variables inicials
let preuOriginal = 120;
let descomptePercentatge = 15;

// Calculem el preu final
let estalvi = (preuOriginal * descomptePercentatge) / 100;
let preuFinal = preuOriginal - estalvi;

console.log("Estalvies: " + estalvi + "€");
console.log("Preu final: " + preuFinal + "€");

Estructures de control: Condicionals i Bucles

Permeten prendre decisions dins del programa i repetir accions tantes vegades com calgui sense duplicar codi.

Estructures clau

  • Selecció (if / else): bifurca el camí segons si una condició és cert o fals.
  • Iteració (while / for): repeteix un bloc mentre es mantingui la condició.

Exemple: classificador de temperatures

let temperatura = 28;

if (temperatura > 30) {
  console.log("Fa molta calor");
} else if (temperatura >= 18) {
  console.log("Temperatura agradable");
} else {
  console.log("Fa fred");
}

// Bucle simple per comptar fins a 3
for (let i = 1; i <= 3; i++) {
  console.log("Volta número: " + i);
}

Programació II: Estructures de dades i POO

La Programació II s'enfoca en l'organització avançada de dades mitjançant llistes, piles, cues i la creació d'objectes reals.

Estructures de dades

Estructures clau que permeten emmagatzemar informació de forma altament eficient per a posteriors cerques i ordenacions.

  • Arrays (Vectors): col·leccions ordenades indexades de zero en endavant.
  • Piles (Stack): LIFO (últim a entrar, primer a sortir).
  • Cues (Queue): FIFO (primer a entrar, primer a sortir).

Exemple de classe i objecte

class Cotxe {
  constructor(marca, model) {
    this.marca = marca;
    this.model = model;
    this.ences = false;
  }

  engegar() {
    this.ences = true;
    console.log(this.marca + " " + this.model + " s'ha engegat!");
  }
}

// Creació d'un objecte a partir de la classe
const elMeuCotxe = new Cotxe("Tesla", "Model 3");
elMeuCotxe.engegar();
