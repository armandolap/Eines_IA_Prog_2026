Programación I (Programación Estructurada)

La programación estructurada consiste en desarrollar programas utilizando estructuras de control simples, evitando código desordenado y favoreciendo la claridad.

Variables

Una variable es un espacio reservado en memoria donde se almacena información. 


// Declaración de variables en C

int edad = 20;
float altura = 1.75;
char letra = 'A';

Tipos de Datos

Los tipos de datos determinan qué clase de información puede almacenar una variable.

  • int
  • float
  • double
  • char
  • bool

Operadores

Los operadores permiten realizar operaciones matemáticas, lógicas y relacionales. 


int suma = 5 + 3;
int resta = 8 - 2;
int mult = 4 * 5;
int div = 20 / 4;

Condicionales

Permiten ejecutar diferentes bloques de código dependiendo de una condición. 


if(edad >= 18)
{
    printf("Mayor de edad");
}
else
{
    printf("Menor de edad");
}

Bucles

Los bucles permiten repetir instrucciones. 


for(int i=0;i<5;i++)
{
    printf("%d",i);
}

Funciones

Una función agrupa instrucciones para reutilizarlas. 


int sumar(int a,int b)
{
    return a+b;
}

Arreglos

Un arreglo almacena varios elementos del mismo tipo. 


int numeros[5]={10,20,30,40,50};

Programación II

En esta materia aparecen estructuras de datos y Programación Orientada a Objetos.

Estructuras de Datos

Son formas organizadas de almacenar información para acceder a ella de forma eficiente.

Listas

Permiten almacenar elementos conectados mediante nodos.

Pilas (Stack)

Funcionan con la regla LIFO (Last In First Out).

Colas (Queue)

Funcionan con la regla FIFO (First In First Out).

Árboles

Organizan información jerárquicamente.

Programación Orientada a Objetos

La POO organiza el software utilizando objetos que contienen datos y comportamientos.

Clase

Es un molde para crear objetos. 


class Persona
{
    String nombre;
}

Objeto

Es una instancia creada a partir de una clase. 


Persona p = new Persona();

Herencia

Permite reutilizar atributos y métodos de otra clase. 


class Alumno extends Persona
{

}

Polimorfismo

Un mismo método puede comportarse de diferentes maneras.

Encapsulamiento

Consiste en proteger los datos de una clase mediante modificadores de acceso. 


private String nombre;
