Programación 1: Programación Estructural

La programación estructural es un estilo de programación que organiza el código en bloques claramente definidos, usando secuencias, condicionales, bucles y funciones.

Definición clave

En Programación 1, el foco está en resolver problemas paso a paso, con funciones o procedimientos que se llaman en un orden lógico y evitan el uso excesivo de saltos incondicionales.

Principios importantes

  • Secuencia: cada instrucción se ejecuta en orden.
  • Selección: se toman decisiones con if y switch.
  • Iteración: se repite código con for, while o do...while.
  • Modularidad: se divide el programa en funciones pequeñas y reutilizables.

Ejemplo en JavaScript: promedio de números

// Función que calcula el promedio de un arreglo de números
function calcularPromedio(numeros) {
  let suma = 0;
  for (let i = 0; i < numeros.length; i += 1) {
    suma += numeros[i]; // Suma cada elemento del arreglo
  }
  return suma / numeros.length; // Devuelve el promedio
}

const datos = [8, 7, 10, 9];
const promedio = calcularPromedio(datos);
console.log('Promedio:', promedio);

En este ejemplo se usa una función con un bucle for para recorrer el arreglo y se devuelve el resultado con return.

Ejemplo en JavaScript: condicionales y bucles

// Función que muestra si una persona es mayor de edad
function verificarEdad(edad) {
  if (edad >= 18) {
    return 'Puede acceder al contenido.';
  } else {
    return 'Necesita autorización de un adulto.';
  }
}

const mensaje = verificarEdad(16);
console.log(mensaje);

// Bucle while para sumar números impares menores a 10
let total = 0;
let contador = 1;
while (contador < 10) {
  total += contador; // Suma el número actual
  contador += 2; // Avanza al siguiente número impar
}
console.log('Suma de impares:', total);

Este código demuestra una estructura de control de flujo con if/else y un bucle while para realizar cálculos repetitivos.

Programación 2: Estructuras de datos y POO

La Programación 2 amplía el enfoque hacia el diseño de estructuras de datos eficientes y la programación orientada a objetos (POO), donde el código se organiza en clases y objetos.

Estructuras de datos

Las estructuras de datos son formas de almacenar y organizar información para poder acceder y manipularla de manera eficiente.

  • Array: colección ordenada de elementos.
  • Lista enlazada: nodos conectados uno a uno.
  • Pila: último en entrar, primero en salir.
  • Cola: primero en entrar, primero en salir.

Ejemplo de POO en JavaScript

// Clase Persona en JavaScript con constructor y método
class Persona {
  constructor(nombre, edad) {
    this.nombre = nombre;
    this.edad = edad;
  }

  saludar() {
    console.log(`Hola, soy ${this.nombre} y tengo ${this.edad} años.`);
  }
}

const persona1 = new Persona('Ana', 23);
persona1.saludar();

Este ejemplo muestra una clase con atributos y un método. Se crea un objeto con new y se invoca su comportamiento.
