# Missió — Web Clínica Dental DEMO

> Document de visió i abast del projecte · Versió catalana monolingüe

---

## visió general

Web corporativa professional per a **Clínica Dental DEMO**, situada a la Plaça del Comte de Sert, 25, Sarrià-Sant Gervasi, 08035 Barcelona

Lloc web estàtic, lleuger, de ràrrega ràpida, que transmeti confiança, professionalitat i proximitat. Dissenyat per atraure pacients nous i facilitar la reserva de visites.

---

## objectius del projecte

| Objectiu | Descripció |
|---|---|
| **Presència digital** | Tenir un web indexat a Google que mostri els serveis de la clínica |
| **Generació de leads** | Facilitar que els visitants demanin cita (WhatsApp + formulari + telèfon) |
| **Credibilitat** | Transmetre confiança mitjançant fotos reals, biografia de la doctora, dades estructurades |
| **Informació útil** | Guia de salut dental per edats que posicioni la clínica com a referent |
| **Rendiment** | Lighthouse >90 a mòbil, Core Web Vitals verds, càrrega <3s |
| **Mantenible** | Stack sense dependències complexes, actualitzable per una sola persona |

## púbic objectiu

- **Pacients potencials** de Barcelona i rodalies (Eixample, Esquerra, Sarrià-Sant Gervasi, Gràcia)
- **Famílies** amb nens petits (odontopediatria)
- **Adults** que necessiten implants, ortodòncia invisible o estètica dental
- **Gent gran** (+65) amb necessitats de rehabilitació oral
- Tots els continguts, texts i etiquetes en **català**

---

## criteris d'èxit

- [ ] Totes les pàgines carreguen en <3s a 4G
- [ ] Lighthouse rendiment >90 (mòbil i desktop)
- [ ] Cap enllaç trencat
- [ ] Formulari de contacte funcional i amb entrega real
- [ ] Google Analytics 4 operatiu amb ID real
- [ ] Sitemap i robots.txt detectats per Google Search Console
- [ ] Dades estructurades JSON-LD vàlides (validar amb Rich Results Test)
- [ ] 100% responsiu a mòbil, tauleta i desktop
