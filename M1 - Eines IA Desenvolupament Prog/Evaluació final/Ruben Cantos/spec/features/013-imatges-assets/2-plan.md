# Plan: Imatges i Assets

## estructura

Totes les imatges a `codi/img/`

## generació

- Fotos reals de la clínica, doctora i tractaments (càmera professional o smartphone d'alta resolució)
- Infografies: eines SVG (Inkscape/Figma) o contractar dissenyador
- Conversió a WebP: eina online (squoosh.app) o script batch
- Favicon: editor SVG inline

## optimització

- Comprimir JPG al 80-85% de qualitat
- WebP amb qualitat 75-80%
- Mantenir aspect ratio exacte
- Afegir atributs `width` + `height` + `loading` + `alt`
