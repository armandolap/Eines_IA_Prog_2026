# Misión
- Producto
Una página web sobre ingresos y gastos de un usuario donde podrán añadirse ingresos y gastos en una fecha concreta y estos datos se irán sumando a un archivo o base datos local.

- Objetivo
a) Que una persona pueda abrir la página
b) Visualizar el "Total" (diferencia entre gastos e ingresos totales)
c) Seleccionar "Gastos" o "Ingresos"
d) Seleccionar "Día", "Semana", "Mes", "Año" o "Período"
e) Añadir un gasto o ingreso clicando en un botón "+":
e.1) Página de "Añadir Gasto/Ingreso": cantidad, selección de Categoría, selección de Fecha, selección de Etiqueta, escribir un Comentario opcional, subir imagen opcional y botón "Añadir" para añadir el gasto o ingreso.
e.1.I) Página de "Añadir Categoría": selección de categoría entre las categorías guardadas en archivo o base de datos local.
e.1.II) Página de "Crear Categoría": Crear una nueva Categoría: Nombre de categoría, seleccionar tipo "Gastos" o "Ingresos", seleccionar símbolo (un icono svg), seleccionar color de fondo para el icono.

- Principios
Tono sobrio y directo.
Se lee igual de bien en el móvil que en el ordenador.
