Ruta del Desarrollador

Fundamentos de programación explicados de forma clara, visual y moderna.

Nivel 1: Programación Estructurada

01. Estructura Secuencial

Es la forma más simple de ejecución. Las instrucciones se ejecutan de manera lineal, una detrás de otra, en el orden estricto en el que están escritas en el archivo fuente.

// Las acciones ocurren en estricto orden
edad = 25;
año_actual = 2026;
año_nacimiento = año_actual - edad;
print(año_nacimiento);

02. Estructuras Condicionales (IF / ELSE)

Permiten al programa bifurcar su flujo de ejecución tomando decisiones lógicas basadas en si una condición se cumple (verdadero) o no (falso).

if (usuario.logeado == true) {
    // Se ejecuta si es verdadero
    mostrar_panel();
} else {
    // Se ejecuta si es falso
    mostrar_login();
}

03. Bucles / Estructuras Cíclicas

Instrucciones diseñadas para repetir un bloque de código múltiples veces mientras una condición lógica específica continúe siendo verdadera.

// Imprime números del 0 al 4
for (let i = 0; i < 5; i++) {
    print("Contador actual: " + i);
}

Nivel 2: Estructuras de Datos y POO

01. Colecciones (Arrays/Listas)

Estructuras de datos que permiten agrupar múltiples elementos del mismo o diferente tipo bajo una sola variable, organizados por un índice numérico.

// Declaración de una lista indexada
let lenguajes = ["Python", "JS", "C++"];

// Acceso mediante índice (base 0)
print(lenguajes[0]); // Salida: Python

02. Clases y Objetos

La base de la POO. Una Clase es la plantilla abstracta (plano de construcción) y el Objeto es la instancia real y concreta nacida a partir de dicha plantilla.

class Coche {
    constructor(marca) {
        this.marca = marca; // Propiedad
    }
}
// Creación del objeto real
let miCoche = new Coche("Tesla");

03. Encapsulamiento

Pilar de la POO que consiste en ocultar el estado interno de un objeto y restringir el acceso directo, exponiendo solo lo necesario a través de métodos públicos.

class CuentaBancaria {
    #saldo = 500; // Propiedad privada (#)

    getSaldo() { 
        return this.#saldo; // Acceso seguro
    }
}
