Programación Nivel 1 y Nivel 2

Guía introductoria sobre Programación Estructurada, Estructuras de Datos y Programación Orientada a Objetos.

Nivel 1 · Programación Estructurada

La programación estructurada es un paradigma que organiza un programa mediante bloques bien definidos, evitando el uso excesivo de saltos incontrolados y facilitando el mantenimiento del código.

Características principales
  • Secuencia de instrucciones.
  • Uso de decisiones.
  • Uso de bucles.
  • Modularización mediante funciones.

Ejemplo de decisión

edad = 20

Si edad >= 18 entonces

    Mostrar "Mayor de edad"

    Mostrar "Menor de edad"

Nivel 2 · Estructuras de Datos

Las estructuras de datos permiten organizar la información para acceder a ella de forma rápida y eficiente.

Estructura Descripción
Array Almacena elementos del mismo tipo.
Lista Permite insertar y eliminar elementos fácilmente.
Pila LIFO (Last In First Out).
Cola FIFO (First In First Out).
Árbol Estructura jerárquica.

Ejemplo de Array

numeros = [10,20,30,40]

Mostrar numeros[0]

Resultado: 10

Ejemplo de Pila

Push(10)

Push(20)

Push(30)

Pop()

Resultado → 30

Nivel 2 · Programación Orientada a Objetos

La Programación Orientada a Objetos (POO) organiza el software mediante objetos que representan entidades reales.

Conceptos fundamentales

  • Clase
  • Objeto
  • Atributos
  • Métodos
  • Encapsulamiento
  • Herencia
  • Polimorfismo
  • Abstracción

Ejemplo de clase

    edad

    Método saludar()

Creación de un objeto
