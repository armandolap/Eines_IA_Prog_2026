What is Programming?

Programming is the process of writing instructions that a computer can follow to perform a task. These instructions are written in a programming language such as Python, Java, or C++.

A program takes input, processes it according to a set of rules, and produces output. The set of rules is called an algorithm.

Key idea: Programming is about breaking a problem down into small, precise steps the computer can execute.

Variables and Data Types

Variables store data in memory so it can be used and manipulated throughout a program. Every variable has a name, a value, and a data type that determines what kind of data it can hold.

Common Data Types

  • Integer — whole numbers (e.g., 42, -7)
  • Float — decimal numbers (e.g., 3.14, -0.5)
  • String — text (e.g., "hello", "42")
  • Boolean — true or false

Example

age = 25            # integer
price = 19.99       # float
name = "Alice"      # string
is_student = True   # boolean

Control Flow

Control flow determines the order in which statements are executed. The three fundamental structures are sequence, selection, and iteration.

Conditionals (Selection)

Conditionals let the program make decisions based on boolean expressions.

score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
else:
    grade = "C"

Loops (Iteration)

Loops repeat a block of code while a condition holds true or for each item in a collection.

# While loop
count = 0
while count < 5:
    print(count)
    count += 1

# For loop
for i in range(5):
    print(i)

Functions

A function is a reusable block of code that performs a specific task. Functions help avoid repetition and make programs easier to read and maintain.

Key Concepts

  • Definition — creating a function with a name and body
  • Parameters — inputs the function accepts
  • Return value — the output the function produces
  • Call — invoking the function to execute it

Example

def greet(name):
    """Return a greeting string."""
    return f"Hello, {name}!"

message = greet("Alice")
print(message)  # Hello, Alice!

Basic Input / Output

Programs communicate with the outside world through input and output (I/O).

Output

Display information to the user, usually on the console or screen.

print("Hello, world!")

Input

Read data from the user via the keyboard or other sources.

name = input("Enter your name: ")
print(f"Welcome, {name}!")
Tip: Always validate user input before using it in your program to avoid unexpected errors.
