Variables

Les variables son contenidors on guardem informacio per reutilitzar-la mes tard. Son el primer que cal entendre per escriure codi.

Que es una variable?

Una variable es un espai amb nom on es guarda un valor (un numero, un text, etc.) per poder-lo recuperar i modificar mes endavant. 

// Guardem la nota d'un examen
let nota = 7;
console.log(nota); // → 7

let i const

Amb let declares una variable que pot canviar. Amb const declares un valor fix que no es pot reasignar. 

let edat = 20;
edat = 21; // podem canviar-lo

const NOM = "Anna";
// NOM = "Pau"; // ❌ Error!

Tipus de dades

Cada variable te un tipus: string (text), number (numero), boolean (cert/fals), array (llista), etc. 

const nom = "Maria";      // → string
const edat = 25;            // → number
const esMajor = true;    // → boolean
const notes = [7, 8, 6];  // → array

Condicionals

Les estructures condicionals permeten que el programa prenga decisions segons si una condicio es certa o falsa.

if / else

Executa un bloc de codi si la condicio es certa; si no, n'executa un altre. 

const nota = 7;

if (nota >= 5) {
  console.log("Aprovat!");
} else {
  console.log("Suspes");
}

else if

Quan hi ha mes de dues opcions, encadenem condicions amb else if. 

const nota = 8;

if (nota >= 9) {
  console.log("Excel·lent!");
} else if (nota >= 7) {
  console.log("Notable!");
} else if (nota >= 5) {
  console.log("Suficient");
} else {
  console.log("Suspes");
}

Operadors de comparacio

Permeten comparar valors: === (igual estricte), !== (diferent), >, <, >=, <=. 

const a = 5;
const b = "5";

console.log(a === b); // false (tipus diferent)
console.log(a !== b); // true
console.log(a  >  b);  // false

Operadors logics

Combinen condicions: && (AND), || (OR), ! (NOT). 

const edat = 20;
const teCarnet = true;

if (edat >= 18 && teCarnet) {
  console.log("Pot conduir");
}

if (edat < 18 || !teCarnet) {
  console.log("No pot conduir");
}

Condicional ternaria

Una forma abreujada d'escriure if / else en una sola línia. 

const nota = 7;

const resultat = nota >= 5
  ? "Aprovat"
  : "Suspes";

console.log(resultat); // → Aprovat

switch

Alternativa a molts else if quan compares una variable amb multiples valors concrets. 

const dia = 3;

switch (dia) {
  case 1:
    console.log("Dilluns"); break;
  case 2:
    console.log("Dimarts"); break;
  case 3:
    console.log("Dimecres"); break;
  default:
    console.log("Altre dia");
}

Bucles

Els bucles repeteixen un bloc de codi multiples vegades. Estalvien escriure el mateix una i altra vegada.

for

Repeteix codi un nombre concret de vegades. Ideal per recórrer arrays o fer una tasca N vegades. 

// Contar del 0 al 4
for (let i = 0; i < 5; i++) {
  console.log("Volta:", i);
}

// Recorrer un array
const noms = ["Ana", "Pau", "Mia"];
for (let i = 0; i < noms.length; i++) {
  console.log(noms[i]);
}

while

Repeteix mentre una condicio sigui certa. Cal assegurar-se que la condicio acabe sent falsa per no crear un bucle infinit. 

let comptador = 0;

while (comptador < 3) {
  console.log("Comptador:", comptador);
  comptador++;
}

// → Comptador: 0
// → Comptador: 1
// → Comptador: 2

for...of

Forma moderna i neta de recórrer arrays sense necessitat d'un index. 

const colors = ["vermell", "blau", "verd"];

for (const color of colors) {
  console.log(`Color: ${color}`);
}

// → Color: vermell
// → Color: blau
// → Color: verd

break i continue

break ix del bucle immediatament. continue salta a la seguent iteracio. 

// break: atura's al 3
for (let i = 0; i < 10; i++) {
  if (i === 3) break;
  console.log(i); // → 0, 1, 2
}

// continue: salta el 2
for (let i = 0; i < 5; i++) {
  if (i === 2) continue;
  console.log(i); // → 0, 1, 3, 4
}

Bucle infinit (perill)

Si la condicio d'un bucle mai es fa falsa, el programa es queda encallat. Vigila sempre la condicio de sortida. 

// ⚠️ Aixo NO s'ha de fer:
// while (true) { }
// Es repetiria per sempre!

// ✅ Correcte: condicio limitada
let x = 0;
while (x < 5) {
  x++;
}

Funcions

Les funcions son blocs de codi reutilitzables que fan una tasca concreta. Les escrius un cop i les crides tantes vegades com calga.

Que es una funcio?

Una funcio agrupa instruccions sota un nom. Quan la crides, s'executen totes les instruccions en ordre. 

// Definicio
function saludar() {
  console.log("Hola, mon!");
}

// Crida
saludar(); // → Hola, mon!
saludar(); // → Hola, mon!

Parametres i arguments

El parametre es la variable que espera la funcio. L'argument es el valor que li passes en cridar-la. 

function saludar(nom) {   // parametre
  console.log("Hola,", nom);
}

saludar("Anna"); // "Anna" es l'argument
saludar("Pau");

Valor de retorn

return fa que la funcio done un resultat que pots guardar en una variable o usar directament. 

function suma(a, b) {
  return a + b;
}

const resultat = suma(3, 4);
console.log(resultat); // → 7

Arrow functions

Forma mes curta d'escriure funcions, molt usada en JavaScript modern. Si te una sola expressio, el return es implicit. 

// Funcio normal
function doble(x) {
  return x * 2;
}

// Arrow function (equivalent)
const doble = (x) => x * 2;

console.log(doble(5)); // → 10

Scope (ambit)

Les variables declarades dins d'una funcio nomes existeixen dins d'ella. Fora no s'hi pot accedir. 

function prova() {
  const missatge = "Dins!";
  console.log(missatge); // → Dins!
}

prova();
// console.log(missatge); // ReferenceError!

Exemple practic: mitjana

Funcio que calcula la mitjana d'un array de notes. Combina funcions, bucles i arrays. 

function calculaMitjana(notes) {
  let suma = 0;
  for (let i = 0; i < notes.length; i++) {
    suma += notes[i];
  }
  return suma / notes.length;
}

const notes = [7, 8, 6, 9];
console.log(calculaMitjana(notes)); // → 7.5
