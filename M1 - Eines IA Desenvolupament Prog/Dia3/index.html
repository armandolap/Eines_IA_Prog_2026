Fonaments de Programació I i II

Conceptes fonamentals de programació amb definicions i exemples pràctics.

Què és la programació?

La programació és el procés d'escriure instruccions que un ordinador pot seguir per realitzar una tasca.

Definició

Programar consisteix a dissenyar, escriure i mantenir el codi font de programes informàtics fent servir llenguatges com JavaScript, Python o Java.

Exemple

console.log('Hola, món!');

Variables i tipus de dades

Les variables emmagatzemen informació a la memòria. Cada variable té un tipus de dada.

Tipus bàsics

  • number: números enters i decimals
  • string: text
  • boolean: vertader o fals

Exemple

let edat = 25;
let nom = 'Anna';
let esEstudiant = true;

Condicionals

Les estructures condicionals permeten executar diferents blocs de codi segons una condició.

Estructures

  • if / else if / else: per a condicions múltiples
  • switch: per a múltiples valors fixos

Exemple

let nota = 85;

if (nota >= 90) {
  console.log('Excel·lent');
} else if (nota >= 70) {
  console.log('Notable');
} else {
  console.log('Suspès');
}

Bucles

Els bucles repeteixen un bloc de codi mentre es compleixi una condició.

Tipus

  • for: quan saps quantes vegades repetir
  • while: mentre es compleixi una condició

Exemple

for (let i = 1; i <= 5; i++) {
  console.log('Número:', i);
}

Funcions

Les funcions agrupen instruccions reutilitzables que realitzen una tasca específica.

Parts d'una funció

  • Declaració: es defineix amb function
  • Paràmetres: valors d'entrada
  • Return: valor que retorna la funció

Exemple

function sumar(a, b) {
  return a + b;
}

console.log(sumar(8, 5));

Arrays

Un array és una col·lecció ordenada d'elements en una sola variable.

Característiques

  • S'hi accedeix per l'índex (comença a 0)
  • Mètodes útils: push, pop, length

Exemple

let fruites = ['poma', 'plàtan', 'taronja'];
console.log(fruites[0]);

POO bàsica

La Programació Orientada a Objectes organitza el codi en classes i objectes.

Conceptes clau

  • Classe: motlle per crear objectes
  • Objecte: instància d'una classe

Exemple

class Persona {
  constructor(nom, edat) {
    this.nom = nom;
    this.edat = edat;
  }
}

let estudiant = new Persona('Lluís', 22);
