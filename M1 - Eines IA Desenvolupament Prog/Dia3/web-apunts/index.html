Fonaments de Programació I

Algorismes i pseudocodi

Un algoritme és una seqüència de passos ordenats per resoldre un problema. El pseudocodi és una manera d'escriure'l en llenguatge humà abans de traduir-lo a un llenguatge de programació.

# Algoritme: calcular la mitjana de dos nombres
Entrada: a, b
suma = a + b
mitjana = suma / 2
Sortida: mitjana

Variables i tipus

Les variables emmagatzemen dades a la memòria. En Python no cal declarar-ne el tipus; s'assigna automàticament segons el valor.

edat = 25               # int
preu = 19.99            # float
nom = "Anna"            # str
es_estudiant = True     # bool

Operadors

Els operadors permeten fer operacions amb valors: aritmètiques, comparacions i lògiques.

a = 10
b = 3

print(a + b)     # 13
print(a % b)     # 1
print(a > b)     # True
print(a > 5 and b < 5)   # True

Entrada / sortida

La funció input() llegeix dades de l'usuari i print() mostra resultats per pantalla.

nom = input("Com et dius? ")
print(f"Hola, {nom}!")

Condicionals

Les estructures condicionals executen un bloc de codi o un altre segons si una condició és certa o falsa.

nota = 85

if nota >= 90:
    print("Excel·lent")
elif nota >= 70:
    print("Notable")
else:
    print("Suspès")

Bucles

Els bucles repeteixen codi mentre es compleixi una condició (while) o per a cada element d'una seqüència (for).

# for
for i in range(5):
    print(i)

# while
comptador = 0
while comptador < 5:
    print(comptador)
    comptador += 1

Funcions

Les funcions agrupen codi reutilitzable. Es defineixen amb def i poden rebre paràmetres i retornar valors.

def sumar(a, b):
    return a + b

def es_parell(n):
    return n % 2 == 0

print(sumar(8, 5))
print(es_parell(8))

Fonaments de Programació II

Llistes i col·leccions

Les llistes emmagatzemen múltiples elements en ordre. S'hi accedeix per índex i tenen mètodes com append(), remove() i len().

fruites = ["poma", "platan", "taronja"]
fruites.append("raim")
print(fruites[0])
print(len(fruites))

for f in fruites:
    print(f)

Cadenes (strings)

Les cadenes són seqüències de caràcters. Admeten operacions com concatenació, indexació i mètodes com upper(), split().

text = "Hola, mon!"
print(text.upper())
print(text[0])
parts = text.split(",")
print(parts)

Recursivitat

Una funció recursiva es crida a si mateixa per resoldre un problema dividint-lo en casos més petits. Cal un cas base per evitar un bucle infinit.

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

print(factorial(5))   # 120

Programació orientada a objectes

La POO organitza el codi en classes (motlles) i objectes (instàncies). Les classes tenen atributs i mètodes.

class Persona:
    def __init__(self, nom, edat):
        self.nom = nom
        self.edat = edat

    def presentar(self):
        return f"Hola, soc {self.nom}"

estudiant = Persona("Lluis", 22)
print(estudiant.presentar())

Fitxers

Llegir i escriure fitxers permet guardar informació de forma persistent. open() obre un fitxer amb un mode determinat (r, w, a).

# Escriure
with open("notes.txt", "w") as f:
    f.write("Hola, mon!")

# Llegir
with open("notes.txt", "r") as f:
    contingut = f.read()
    print(contingut)

Maneig d'errors

Les excepcions permeten gestionar errors d'execució sense que el programa es trenqui. Es fan servir try / except.

try:
    num = int(input("Introdueix un numero: "))
    resultat = 10 / num
    print(resultat)
except ValueError:
    print("No es un numero valid")
except ZeroDivisionError:
    print("No es pot dividir per zero")
