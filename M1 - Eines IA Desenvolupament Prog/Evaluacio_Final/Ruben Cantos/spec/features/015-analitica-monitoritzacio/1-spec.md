# Feature: Analítica i Monitorització

Configuració de Google Analytics 4 i Google Search Console.

## abast

1. **Google Analytics 4:**
   - Comptar ID real GA4
   - Afegir script `gtag.js` a totes les pàgines
   - Configurar esdeveniments:
     - `click_whatsapp` (clic al botó flotant)
     - `submit_contact_form` (enviament formulari)
     - `click_phone` (clic al telèfon)
   - `anonymize_ip: true`
2. **Google Search Console:**
   - Verificar propietat del domini (DNS/TXT)
   - Enviar `sitemap.xml`
   - Monitoritzar impressions, clics, posició mitjana
3. **Google Business Profile:**
   - NAP (Name, Address, Phone) consistent amb el web
   - Enllaçar des del web (schema.org + link al footer)
