# Sobre el proyecto

Un par de observaciones sobre esta entrega

- Me salí un poco del guión. Dada la estructura para la especificación y las instrucciones para realizarla que vimos en clase, procedí a utilizar gemini para generar la especificación buscando una descripción más detallada del producto final en términos de elementos de las páginas y su organización interna. También busqué que las instrucciones al agente para que siguiera un proceso más determinístico y repetible fuesen en lo posible explicitas y detalladas

- Tengo problemas para instalar opencode en mi ordenador por compatibilidad de versiones con el sistema operativo. Por eso en la intrega no incluyo el código generado. Veré de correr la especificación en el ordenador de clase y comparar los resultados.
