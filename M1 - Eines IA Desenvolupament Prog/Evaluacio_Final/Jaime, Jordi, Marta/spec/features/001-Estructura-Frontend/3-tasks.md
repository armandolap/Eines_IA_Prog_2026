# Tasques — 001 Història de Barcelona
Ordre d'execució. Marca cada tasca en completar-la.

[x] T1 — Esquelet. Crear index.html amb <head> i <body>, crear estils.css i enllaçar-lo des de totes les pàgines del projecte.
[x] T2 — Estils base. Configurar estils.css seguint la skill Disseny-web (mode fosc, paleta i tokens), sense utilitzar colors fora de la guia.
[x] T3 — Capçalera i navegació. Maquetar una capçalera sticky amb el títol "Història de Barcelona", el subtítol "Un viatge a través dels segles", el menú de navegació i el botó hamburguesa amb aria-label.
[x] T4 — Menú responsiu. Implementar el JavaScript mínim perquè el menú s'obri i es tanqui al mòbil amb el botó hamburguesa, es tanqui en seleccionar un enllaç i sigui horitzontal i visible en pantalles grans.
[x] T5 — Hero i índex. Crear el hero amb fons slate i un patró CSS inspirat en un mosaic romà (opus tessellatum). Afegir la graella de targetes amb numeració romana, connexió visual de línia del temps i enllaç a cadascuna de les pàgines dels segles.
[x] T6 — Pàgines dels segles. Crear les vuit pàgines (segleII.html, segleV.html, segleVIII.html, segleXII.html, segleXV.html, segleXVIII.html, segleXIX.html i segleXX.html) amb el títol corresponent, un mínim de 10 línies de contingut històric, una fotografia representativa (placeholder) amb peu de foto i l'enllaç de retorn a index.html.
[x] T7 — Cronologia i navegació. Afegir a totes les pàgines de segle una barra cronològica horitzontal amb el segle actiu ressaltat i els enllaços de navegació anterior/següent.
[x] T8 — Accessibilitat i detalls. Assegurar focus visible, contrast suficient, responsive (1 → 2 → 3 columnes), efectes hover subtils i qualsevol detall visual definit a la spec.
[x] T9 — Verificació. Comprovar que els enllaços porten a la pàgina corresponent de cada segle, que el menú funciona correctament al mòbil, que el disseny segueix els tokens de Disseny-web i que es compleixen tots els criteris d'acceptació.
