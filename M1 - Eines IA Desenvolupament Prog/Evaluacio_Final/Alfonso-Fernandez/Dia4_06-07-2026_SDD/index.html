Programación

La programación es el proceso de escribir instrucciones que un ordenador puede entender y ejecutar. Un programa es una secuencia de órdenes que resuelven un problema.

Conceptos clave

  • Algoritmo: conjunto de pasos ordenados para resolver un problema.
  • Código fuente: texto escrito en un lenguaje de programación.
  • Compilador / Intérprete: traduce el código a instrucciones de máquina.
// Ejemplo: hola mundo
console.log("Hola, mundo!");

Variables

Una variable es un espacio con nombre dentro de la memoria del ordenador donde se guarda un valor. El valor puede cambiar durante la ejecución del programa.

Tipos básicos en JavaScript

  • string: texto (ej: "Hola")
  • number: números enteros o decimales
  • boolean: true o false
let nombre = "Ana";
let edad = 20;
let esEstudiante = true;
console.log(nombre, edad, esEstudiante);

Condicionales

Las estructuras condicionales permiten ejecutar un bloque de código u otro según si una condición es cierta (true) o falsa (false).

Estructuras principales

  • if / else if / else
  • switch: múltiples casos
  • Operador ternario: cond ? a : b
let nota = 7;
if (nota >= 5) {
  console.log("Aprobado");
} else {
  console.log("Suspenso");
}

Bucles

Los bucles repiten un bloque de código mientras una condición sea cierta. Se usan para recorrer listas o repetir operaciones.

Bucles más comunes

  • for: repite un número conocido de veces
  • while: repite mientras la condición sea cierta
  • do...while: ejecuta al menos una vez
for (let i = 1; i <= 5; i++) {
  console.log("Vez " + i);
}

Funciones

Una función es un bloque de código reutilizable que realiza una tarea concreta. Puede recibir datos de entrada (parámetros) y devolver un resultado.

Formas de definir funciones

  • Función declarada: function sumar(a, b) { ... }
  • Función flecha: const sumar = (a, b) => a + b
function saludar(nombre) {
  return "Hola, " + nombre + "!";
}
console.log(saludar("María"));

Arrays

Un array es una estructura de datos que almacena múltiples valores en una sola variable. Cada elemento tiene una posición (índice) que comienza en 0.

Operaciones habituales

  • Acceder: array[0]
  • Añadir: array.push(valor)
  • Recorrer: for o forEach
let colores = ["rojo", "verde", "azul"];
colores.push("amarillo");
for (let c of colores) {
  console.log(c);
}

Programación Orientada a Objetos (POO)

La POO organiza el código en objetos que contienen datos (propiedades) y comportamiento (métodos). Una clase es la plantilla para crear objetos.

Conceptos fundamentales

  • Clase: define la estructura (class)
  • Objeto: instancia de una clase (new)
  • Constructor: inicializa las propiedades
class Gato {
  constructor(nombre) {
    this.nombre = nombre;
  }
  saludar() {
    console.log("¡Miau! Soy " + this.nombre);
  }
}
let gato = new Gato("Bigotes");
gato.saludar();
