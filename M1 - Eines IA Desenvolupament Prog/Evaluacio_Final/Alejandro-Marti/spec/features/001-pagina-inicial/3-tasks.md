# 001 — Tasques de desenvolupament: Pàgina inicial

Ordre recomanat d'execució. Marca cada tasca en completar-la.

- [x] T1 — Preparació base. Crear l'estructura inicial del projecte amb l'arxiu HTML principal i els fitxers d'estil i script necessaris, i enllaçar-los correctament.
- [x] T2 — Estructura HTML semàntica. Definir el header, el hero, el menú de navegació, la introducció i les seccions temàtiques amb identificadors clars per als enllaços d'àncora.
- [x] T3 — Contingut inicial. Redactar el text introductori, els títols de cada secció i els resums finals amb un llenguatge clar, coherent i adequat al tema.
- [x] T4 — Maquetació base. Organitzar la disposició general de la pàgina amb espaiat, jerarquia visual i separació clara entre introducció i seccions de contingut.
- [x] T5 — Disseny visual del tema. Aplicar una paleta coherent, tipografia legible, cards de dades i elements visuals que donin un aspecte professional i homogeni.
- [x] T6 — Capçalera i navegació principal. Implementar el header sticky, el títol, la descripció breu i el menú amb enllaços interns ben organitzats.
- [x] T7 — Menú responsive. Afegir el botó hamburguesa, l'obertura i tancament al mòbil, el tancament en seleccionar una secció i l'estat actiu del menú.
- [x] T8 — Visualització de dades. Construir les seccions de temperatura, humitat, pressió, vent, sol, aire, lluna, aigua i residus amb una presentació visual clara i atractiva.
- [x] T9 — Animacions i ambientació. Incorporar el sol animat segons el scroll, elements decoratius d'ambient i transicions suaus que reforcin la narrativa sense saturar el contingut.
- [x] T10 — Integració 3D. Afegir la skill 3d-web-experience al final de la pàgina per mostrar un cub 3D que resumeixi les dades més rellevants.
- [x] T11 — Accessibilitat i UX. Revisar contrast, focus visible, aria-label del botó del menú, ordre de lectura, mida del text i navegació amb teclat.
- [x] T12 — Fes que el cub 3D sigui interactiu, es a dir, que el lector pugui controlar la rotacio del cub amb el moviment del cursor o 
en cas del mobil amb el tactil.
- [x] T13 — Afegeix al aprtatat de les fases lunars una animació interactva senzilla mostrant les diferents fases en ordre.
- [x] T14 — Validació final. Comprovar que els enllaços funcionen, el menú respon bé en desktop i mòbil, el disseny és coherent i la pàgina és fàcil de navegar.
